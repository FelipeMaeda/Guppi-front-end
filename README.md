# Guppi-front-end

Guppi-front-end is a front-end development of the project Guppi.


![alt-text](prototype.gif)

### Installation:

Clone this repo and install the dependencies.

```sh
git clone https://github.com/FelipeMaeda/GuppiApp.git
cd GuppiApp
npm install
```

### Usage:

Start the server

```sh
npm start
```
